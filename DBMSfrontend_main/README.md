# DBMS Frontend

A React frontend for the DBMS backend.

## Setup

1. Open terminal in `DBMSfrontend_main`
2. Run `npm install`
3. Run `npm run dev`

The app runs on `http://localhost:5500` to match backend CORS settings.

## Notes

- Admin can create students, professors, courses, and timetable entries.
- Professors can load student enrollments, update marks, and post announcements.
- Students can view their profile, courses, announcements, and timetable, and register for a course by ID.
